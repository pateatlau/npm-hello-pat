Hello World NPM Module
=========

npm hello world module

## Installation

  `npm install npm-hello-pat`

## Usage

    var npmHelloPat = require('npm-hello-pat');

    var hello = npmHelloPat('world');
  
  
  Output should be `Hello, world!`
